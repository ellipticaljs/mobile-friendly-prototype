MOBILE FRIENDLY PROTOTYPE
===========================

A responsive, web-components starter-kit.


##Install

``` bash

git clone https://github.com/ellipticaljs/mobile-friendly-prototype.git
mv mobile-friendly-prototype my-project
cd my-project
npm install
bower install

```

##Tasks

``` bash
gulp start-live

```

##Browser

``` bash
http://localhost:9040/

```
